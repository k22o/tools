## 法令規則・規則

- 日本
  - **サイバーセキュリティ基本法**
    - 日本のサイバーセキュリティ政策の基本理念を定め、国・地方公共団体・重要インフラ事業者等の責務や、国が講ずべき施策の基本的な枠組みを規定する法律
  - **個人情報の保護に関する法律 (個人情報保護法)**
    - 個人情報を取り扱う事業者が、その取得・利用・第三者提供・安全管理などにおいて遵守すべきルールを定めた法律
  - **不正アクセス行為の禁止等に関する法律 (不正アクセス禁止法)**
    - 他人のID・パスワードを無断で使用してアクセスする行為や、それを助長する行為 (フィッシング目的でのID・PW取得等) を禁止する法律
- EU圏
  - **GDPR (General Data Protection Regulation / 一般データ保護規則)**
    - EU域内の個人データ保護に関する規則。EU域内に住む人の個人データを扱う事業者は、EU域外の企業であっても適用対象となる (域外適用) ことが特徴
- 国際基準
  - **ISO/IEC 27001**
    - 組織における情報セキュリティマネジメントシステム (ISMS) の構築・運用・維持・改善に関する要求事項を定めた国際規格
  - **NIST SP 800**
    - 米国国立標準技術研究所 (NIST) が発行する情報セキュリティに関するガイドライン・ベストプラクティス集のシリーズ。アクセス制御やインシデント対応など個別テーマごとに文書が分かれている

## その他

- **リスクベース認証**
  - 普段と異なる状況でのアクセスがあった場合には追加の認証を要求することで、不正アクセスに対する安全性を高める
