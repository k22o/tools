# レビューの種類

- アドホックレビュー
  - 近くの同僚などに質問するというような簡単な確認
- ピアレビュー
  - 成果物を対象に、同僚間で確認を実施する
  - リアルタイムまたはオンデマンドで行われます
- パスアラウンド
  - 成果物を順次メンバーに渡して非同期にレビューを行う
- ウォークスルー
  - 作成者がファシリとなって関係者が集まって議論しながらチェックする
- インスペクション
  - 事前に定められた観点で第三者が厳密に点検して、欠陥の指摘や認定を行う
