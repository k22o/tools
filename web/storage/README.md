## localStorage
- webブラウザ上にデータを保存する
- セッションなどを跨いでも保持される
- 永続的な保持は、バグが生じる場合もあるので注意
- 気密性の高い情報の保持には不向き ([reference](https://techracho.bpsinc.jp/hachi8833/2019_10_09/80851 )

## sessionStorage
- localStorageとほぼ同様
- こちらは、セッションを跨いでは保持されない
