# ネットワーク

## OSI参照モデル

通信機能を7つの階層に分割し、各層の役割を標準化したモデル。

| 層番号 | 層名 | 役割 |
| --- | --- | --- |
| L7 | アプリケーション層 | 具体的な通信サービス (メール・Web等) をアプリケーションに提供する |
| L6 | プレゼンテーション層 | データの表現形式 (文字コード・暗号化・圧縮等) を変換する |
| L5 | セッション層 | 通信の開始から終了までのセッションを確立・維持・切断する |
| L4 | トランスポート層 | 通信の信頼性を確保し、データを正しい相手先アプリケーションまで届ける |
| L3 | ネットワーク層 | 異なるネットワーク間で、宛先までの経路選択とアドレッシングを行う |
| L2 | データリンク層 | 直接接続された機器間で、誤り検出等を行いながらデータを転送する |
| L1 | 物理層 | 電気信号や光信号など物理的な伝送媒体を通じてビット列を送受信する |


## TCP/IPにおけるプロトコル

### 階層 

TCP/IPは、OSI参照モデルの7階層を実装に即して4階層に集約したモデル。

| TCP/IPの階層 | 対応するOSI参照モデルの層 | 役割 |
| --- | --- | --- |
| (4) アプリケーション層 | L5 セッション層 / L6 プレゼンテーション層 / L7 アプリケーション層 | 具体的な通信サービスをアプリケーションに提供する |
| (3) トランスポート層 | L4 トランスポート層 | 通信の信頼性を確保し、データを正しい相手先アプリケーションまで届ける |
| (2) インターネット層 | L3 ネットワーク層 | 異なるネットワーク間で、宛先までの経路選択とアドレッシングを行う |
| (1) ネットワークインターフェース層 | L1 物理層 / L2 データリンク層 | 直接接続された機器間で、物理的な信号の送受信とデータの転送を行う |
